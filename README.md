> convert markdown to html


## Install

```sh
$ npm install --save broccoli-html
```


## Usage

### JavaScript

```js
var broccoli-html = require('broccoli-html');

broccoli-html('Rainbow');
```

### Command Line reference

```sh
$ npm install --global broccoli-html
$ broccoli-html --help
```

## Run Test
```sh
npm test
```
